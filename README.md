## 書籍の概要

多くのサンプルで、PythonとRustのプログラムを比較して、PythonプログラマーがRustに親しむことができるように配慮しています。難しいと言われるRustを気軽にはじめられる本です。

Rust学習の難所「所有権システム」も「ジェネリクス」も豊富なサンプルで楽々マスターしましょう。
